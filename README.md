# Hipparcos HR Diagram

This repository supports an introduction to astrophysics course for physics majors homework exercise in which students construct Hertzsprung-Russell diagrams using real stellar data from the Hipparcos space telescope mission. Students write Python code to compute absolute magnitudes from parallax measurements and generate HR diagrams for both apparent-magnitude-limited and distance-limited samples of stars.

## Running in a GitHub Codespace

1. On the repository page, click **Code > Codespaces > Create codespace on master**.
2. Wait for the codespace to finish building and loading.
3. In the file explorer, open `make_hr_diagram.ipynb`.
4. When prompted to select a kernel, choose the default **Python** environment.
5. Run the notebook: **Run > Run All Cells** (or `Shift+Enter` to step through cells one at a time).

## Closing a Codespace

Go to [github.com/codespaces](https://github.com/codespaces), click **...** next to your codespace, and select **Stop codespace**.

> **Warning:** Do NOT select **Delete** — this permanently destroys all your work with no recovery.

## Re-opening a Codespace

Go to [github.com/codespaces](https://github.com/codespaces) from any browser and click your codespace to reopen it. Do not create a new codespace, as it will start fresh with no saved work.
